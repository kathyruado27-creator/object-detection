# 🎥 Live Object Detection & Tracing

A real-time object detection and tracking web app built with:

- Streamlit
- Streamlit WebRTC
- YOLOv8 (Ultralytics)
- OpenCV

The application uses your webcam to detect and track objects live in the browser.

---

# 📌 Features

✅ Real-time webcam streaming  
✅ YOLOv8 object detection  
✅ Object tracking with tracing  
✅ Fast and lightweight interface  
✅ Browser-based UI using Streamlit
